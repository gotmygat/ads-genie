operated.

inside deterministic
infrastructure

workflow. MCP tools execute

Routing is infrastructure, not
AI

Autonomy levels determine approval
paths. The LLM does not decide its

Every decision must be
replayable

Security Architecture
Security is not bolted on, it is designed into every layer of the system from
day one.
Three-Point AI Firewall
Validate inbound (Slack
messages), outbound (LLM-
proposed actions against
autonomy config), and external
(Google Ads API responses)
before any processing occurs.
Separate IAM Roles
Distinct roles for the monitoring
Lambda, action-execution
Lambda, and Slack handler.
Least-privilege enforced at the
infrastructure level, not by
convention.
Zero Hardcoded Secrets
All credentials; Google OAuth, Slack tokens, Claude API key are stored
in AWS Secrets Manager. No secrets in code, environment variables, or
configuration files.
